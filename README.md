# EcoRide

EcoRide is a modern, sustainable ride-sharing platform that connects drivers and passengers heading in the same direction, enabling cost-effective and environmentally friendly travel. By choosing shared rides through EcoRide, users contribute to reduced traffic congestion, lower carbon emissions, and a positive environmental impact.

## Features

- **Eco-friendly Travel**: Promotes shared rides to minimize environmental impact.
- **Cost Savings**: Reduces costs for drivers and passengers by sharing expenses.
- **Community-Oriented**: Connect passengers with drivers heading in the same direction.
- **Responsive Design**: Optimized for various screen sizes.
- **Convenient Links**: Easy navigation to Home, About Us, and Contact pages.

---

## Purpose and Philosophy

EcoRide exists with a clear mission: to make travel more accessible, convenient, and sustainable. By embracing the values of resource-sharing and community-building, EcoRide seeks to offer an efficient alternative to traditional transportation methods. Every shared ride is a step toward less traffic, reduced carbon emissions, and a sustainable future.

## Getting Started

To get started with EcoRide, you must clone the repository, install its dependencies, and set up the environment for your development needs.

### Prerequisites
Ensure the following tools and software are installed on your system:
- A web server (e.g., Apache, Nginx)
- PHP (at least version 7.4)
- A database management system (e.g., MySQL)

### Install Dependencies
Install any dependencies outlined in the included project composer file or guidelines.

---

## Installation and Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/gcsoares24/EcoRide.git
   ```
2. **Move to the Project Directory**
   ```bash
   cd EcoRide
   ```
3. **Configure Your Environment**
   - Set up a `.env` file for environment variables.
   - Add database connection details.

4. **Start Your Development Environment**
   - Use a PHP server or custom Docker configuration as applicable.

5. **Migrate the Database**
   ```bash
   php artisan migrate
   ```

## Usage

- Access the main landing page via your browser.
