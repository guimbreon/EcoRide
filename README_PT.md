# EcoRide

EcoRide é uma plataforma moderna e sustentável de partilha de boleias que liga condutores e passageiros que seguem na mesma direção, permitindo viagens económicas e amigas do ambiente. Ao escolher boleias partilhadas através do EcoRide, os utilizadores contribuem para a redução do congestionamento do trânsito, a diminuição das emissões de carbono e um impacto ambiental positivo.

## Funcionalidades

* **Viagens Ecológicas**: Promove a partilha de boleias para minimizar o impacto ambiental.
* **Poupança de Custos**: Reduz os custos para condutores e passageiros através da partilha de despesas.
* **Orientado para a Comunidade**: Liga passageiros a condutores que seguem na mesma direção.
* **Design Responsivo**: Otimizado para diferentes tamanhos de ecrã.
* **Ligações Convenientes**: Navegação simples para as páginas Início, Sobre Nós e Contactos.

---

## Propósito e Filosofia

O EcoRide existe com uma missão clara: tornar as viagens mais acessíveis, convenientes e sustentáveis. Ao adotar os valores da partilha de recursos e da construção de comunidade, o EcoRide procura oferecer uma alternativa eficiente aos métodos de transporte tradicionais. Cada boleia partilhada é um passo em direção a menos trânsito, menos emissões de carbono e um futuro mais sustentável.

## Primeiros Passos

Para começar a utilizar o EcoRide, é necessário clonar o repositório, instalar as dependências e configurar o ambiente de desenvolvimento.

### Pré-requisitos

Certifica-te de que tens os seguintes softwares instalados no teu sistema:

* Um servidor web (por exemplo, Apache ou Nginx)
* PHP (versão 7.4 ou superior)
* Um sistema de gestão de base de dados (por exemplo, MySQL)

### Instalar Dependências

Instala todas as dependências indicadas no ficheiro `composer` do projeto ou nas respetivas instruções.

---

## Instalação e Configuração

1. **Clonar o Repositório**

   ```
   git clone https://github.com/guimbreon/EcoRide.git
   ```

2. **Mover para o Diretório do Projeto**

   ```
   cd EcoRide
   ```

3. **Configurar o Ambiente**

   * Criar um ficheiro `.env` para as variáveis de ambiente.
   * Adicionar os dados de ligação à base de dados.

4. **Iniciar o Ambiente de Desenvolvimento**

   * Utilizar um servidor PHP ou uma configuração personalizada com Docker, se aplicável.

5. **Migrar a Base de Dados**

   ```
   php artisan migrate
   ```
